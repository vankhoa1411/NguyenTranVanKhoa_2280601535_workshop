## Introduction

Throughout the **12-week internship**, I participated in a training program and hands-on practice with **Amazon Web Services (AWS)**. The internship followed a learning roadmap starting from the fundamentals of cloud computing, AWS infrastructure management, security, networking, and finally building and deploying the **Cloud-Native AI Document Extractor & Analytics** project using a Serverless architecture.

The activities were recorded and summarized on a weekly basis to monitor learning progress, practical experience, as well as the achievements accomplished throughout the internship.

### Weekly Internship Activities

**Week 1:** [Getting Started with AWS, Creating an AWS Account, and Practicing Basic EC2 and S3](1.1-week1/)

**Week 2:** [Studied Module 01-01: Introduction to AWS, Learned Amazon EC2, Amazon S3, Amazon RDS, and Cloud Computing Models](1.2-week2/)

**Week 3:** [Studied Module 01-02: AWS Management Console, Practiced Creating and Managing Amazon EC2 Instances](1.3-week3/)

**Week 4:** [Studied Module 01-03: Generative AI on AWS, Learned Amazon Bedrock and AWS AI Services](1.4-week4/)

**Week 5:** [Studied Module 01-04: Cost Optimization, Learned AWS Billing, Cost Explorer, and AWS Free Tier](1.5-week5/)

**Week 6:** [Advanced Amazon EC2 Practice, Configured Security Groups, Connected via SSH, and Deployed a Website](1.6-week6/)

**Week 7:** [Learned Amazon S3, Managed Buckets, Configured Access Permissions, and Deployed a Static Website](1.7-week7/)

**Week 8:** [Learned AWS IAM, Practiced Managing Users, Groups, Roles, Policies, and Security Best Practices](1.8-week8/)

**Week 9:** [Learned Amazon VPC, Configured Subnets, Internet Gateway, Route Tables, and AWS Networking Architecture](1.9-week9/)

**Week 10:** [Analyzed Requirements and Designed the Serverless Architecture for the Cloud-Native AI Document Extractor & Analytics Project](1.10-week10/)

**Week 11:** [Developed the System, Integrated Amazon S3, AWS Lambda, and Amazon Textract for Automated Document Processing](1.11-week11/)

**Week 12:** [Completed the Project, Performed System Testing, Summarized the Results, and Finished the Internship Report](1.12-week12/)
