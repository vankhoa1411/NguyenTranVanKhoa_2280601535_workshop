Tại đây sẽ là phần liệt kê, giới thiệu các blogs mà các bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). Ví dụ:

###  [Blog 1 - Triển khai MuleSoft Runtime Fabric trên ROSA](3.1-Blog1/)
Blog này giới thiệu mô hình triển khai MuleSoft Runtime Fabric (RTF) trên cụm Red Hat OpenShift Service on AWS (ROSA), giúp doanh nghiệp xây dựng một nền tảng tích hợp Hybrid Cloud hiện đại, đồng bộ hóa và quản trị API một cách dễ dàng và tối ưu chi phí.

###  [Blog 2 - Giám sát VPN thông minh: Tự động phân tích AWS Site-to-Site VPN Logs bằng AI](3.2-Blog2/)
Blog này giới thiệu phương pháp tự động hóa thu thập và quản lý BGP/IKE VPN logs lên Amazon CloudWatch Logs, đồng thời sử dụng mô hình AI tạo sinh trên Amazon Bedrock kết hợp Lambda Serverless để phân tích nguyên nhân gốc rễ (RCA) của các sự cố mạng VPN nhanh chóng.

###  [Blog 3 - Tự động hóa giao diện AI Agent: Chuẩn hóa Generative UI bằng giao thức AG-UI trên Amazon Bedrock AgentCore](3.3-Blog3/)
Blog này giới thiệu cách chuẩn hóa giao tiếp giữa backend AI và frontend UI bằng giao thức mở AG-UI (Agent-User Interaction Protocol) trên Amazon Bedrock AgentCore Runtime, mở ra khả năng xây dựng các AI Agent hiện đại với Generative UI, trạng thái đồng bộ và tương tác người dùng (Human-in-the-loop).
